Packing/unpacking

If you want to transfer a project from one computer to another then it must be packed into a .ZIP file.


Packing a project


  1. Click on 'Pack project...' from the File menu.


  1. Enter a file name for the .ZIP file and click on Save.


  1. You can now unpack the .ZIP file on another computer where Memoranda is installed.



Unpacking a project


  1. Click on 'Unpack project...' from the File menu.


  1. Enter a file name for the .ZIP file and click on Open.


  1. You can now work on the project.
