Requirements

To build and run Memoranda you need Java 2 Runtime Environment (JRE) (version 1.4.0 onwards). You can download the JRE for a number of platforms from http://java.sun.com. Recommended is version 1.4.2.

The "System tray icon" feature is supported only on

  • Microsoft Windows
  • K Desktop Environment (KDE) 3.0 and higher

To build Memoranda you will also need the Jakarta Ant build tool (v1.5 onwards). You can download Ant from http://jakarta.apache.org.
