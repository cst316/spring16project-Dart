Changing project properties

  1. Click on the symbol to view the list of projects.


  1. Click on a project to select it.


  1. Left-click and select Project properties.



  1. The following box will appear:



Change the project name, start date, etc in the box which appears. Finally click on Ok.


If you know the end date click on the checkbox so you can edit it as well.


You can also select the date from a calendar by clicking the calendar icon after the date box.
