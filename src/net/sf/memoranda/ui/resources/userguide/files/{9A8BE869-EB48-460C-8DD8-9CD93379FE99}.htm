Creating a project

  1. Click on the project icon at the top right end of the window and select new project (Alternatively you may click on New Project from the File menu).


The following box will appear:



  1. Enter a title, start date and click on 'Ok'.  If you know the end date click on the checkbox first.


You can also select the date from a calendar by clicking the calendar icon after the date box.


  1. Start creating tasks, events, etc for this project.
