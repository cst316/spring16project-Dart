Basics (what? why?)

What are projects?


A project is an activity you need to organize, it has a start date and an end date. For example, a student may have projects like 'exam preparation', 'excercise schedule', etc.


Each project has its own agenda, events, tasks, etc. Projects are completely independent of each other and so take care not to schedule different events for the same time!



Why use projects?


Organizing your schedule into seperate projects is a great idea. For example, you can have seperate schedules for working and entertainment. Also, software developers often work on a number of projects that have need to have seperate schedules.
