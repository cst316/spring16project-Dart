Installation

  1. Unpack the downloaded zip file into a directory of your choice. A sub-directory named 'memoranda**-**' will be created, this folder will be known as the "Memoranda home directory".


  1. If you have downloaded a dependencies package in the separate zip file, unpack it into the Memoranda home directory (after unzipping make sure that the 'lib' sub-directory is not empty).


  1. Start Memoranda
