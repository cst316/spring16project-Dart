Starting Memoranda

Important notes:


Linux users please read

FOR KDE USERS: You can put the systray4jd daemon in your autostart and

remove its startup line from memoranda.sh. You also can download and install

systray4j as a shared library (*.so) but currently its unstable and cannot be recommended.


NOTE FOR NON-KDE UNIX USERS: You can safely remove the systray4j startup line

from memoranda.sh to avoid any unknown but possible nasties.


Launching in minimized mode

To launch Memoranda in minimized mode (system tray only), use the "-m"

option. You can also set this behavior in File->Preferences menu.



Starting Memoranda


  1. Windows:  Make sure that JRE executables directory (jre/bin) are on your system PATH

Linux: Make memoranda.sh and lib/kde/systray4jd executable


  1. Go to the Memoranda home directory (the directory where Memoranda is installed).
  2. Run 'memoranda.bat' (on Windows) or './memoranda.sh' (on Unix).
  3. Start using Memoranda
